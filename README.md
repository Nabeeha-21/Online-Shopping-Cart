# 🛒 Online Shopping Cart

## 📌 Project Overview

The Online Shopping Cart is a Java console-based application developed using Object-Oriented Programming (OOP) concepts. It allows users to add products to a shopping cart, view the items in the cart, and calculate the total purchase amount through a simple menu-driven interface.

This project was developed to practice Java Collections such as **ArrayList** and **HashMap**, along with classes, objects, and user input handling.

---

## ✨ Features

- Add products to the shopping cart
- View all products in the cart
- Manage product quantities using HashMap
- Store product details using ArrayList
- Calculate the total price of all products
- Simple menu-driven console interface

---

## 🛠️ Technologies Used

- Java
- Object-Oriented Programming (OOP)
- ArrayList
- HashMap
- Scanner Class

## 🚀 How to Run

1. Clone the repository.
2. Open the project in any Java IDE (Eclipse, IntelliJ IDEA, or VS Code).
3. Compile all the Java files.
4. Run the `Main.java` file.
5. Follow the menu displayed in the console.
